# @devalbo-cli/ui

Isomorphic UI primitives and shell providers for terminal/browser command UIs.

## Exports

- primitives: `TextInput`, `Spinner`, `Select`, `TreeView`, `StatusBar`, `ScrollArea`
- shell: `ShellContext`, `BrowserShellProvider`, `TerminalShellProvider`, `InteractiveShell`
- hooks: `usePlatform`, `useKeyboard`
